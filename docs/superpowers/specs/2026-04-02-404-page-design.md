# 404 Page — Design Spec

**Date:** 2026-04-02  
**Status:** Approved

---

## Overview

A fun, retro-themed 404 error page for Michel Encarnación's portfolio. Features a pixel art T-Rex (drawn entirely in CSS using box-shadows — no external images) that runs in and destroys the "404" text, followed by a humorous message and a button back to the home page.

---

## File

- **Location:** `src/pages/404.astro`
- **Approach:** Astro page with inline `<style>` CSS — no React, no client-side JS runtime required

---

## Layout

Full-screen centered layout (`min-h-screen flex flex-col items-center justify-center`) with three vertical zones:

1. **404 number** — large Sora bold heading, styled to look "destroyed" after the dino hits it
2. **Pixel art T-Rex** — CSS-only pixel art using `box-shadow` technique, centered below the number
3. **Message + CTA** — humorous Spanish text + green accent button linking to `/`

Uses the existing `Layout.astro` wrapper for consistent `<head>`, fonts, and dark theme.

---

## Animation Sequence

| Time | Event |
|------|-------|
| 0.0s | "404" appears static, full opacity |
| 0.5s | T-Rex enters running from the right (`@keyframes dino-enter`) — legs cycle (walk animation) |
| 0.8s | T-Rex reaches center — "404" shakes violently (`@keyframes shake`) then letters scatter/explode (`@keyframes explode` — translate + rotate + fade) |
| 1.0s | T-Rex holds "roar pose" — tail wags in loop (`@keyframes tail-wag`) |
| 1.2s | Message and button fade in (`@keyframes fade-up`) |

All animations are CSS-only `@keyframes`. No JavaScript required.

---

## Pixel Art T-Rex

Drawn using the CSS `box-shadow` technique on a single `<div>`:
- One `div` acts as the "pixel canvas" — each pixel is a `box-shadow` offset with color
- Size: 24×24 pixel grid, scaled up with `transform: scale(4)` (renders at ~96px base, visually large)
- Primary color: `#4ade80` (accent green) for the body
- Darker green (`#16a34a`) for shading/outline pixels
- Eye: white pixel with dark pupil
- Two frames for the walk cycle (legs alternate), swapped via `animation`

---

## Aesthetics

- **Background:** `#0a0a0a` (existing dark theme `bg-bg-primary-dark`)
- **Scanlines overlay:** Subtle repeating-linear-gradient CSS pattern for retro 8-bit feel
- **404 text:** Sora font, `14rem` desktop / `7rem` mobile, accent green — fragments with `transform` on shake
- **Message text:** Sora, humorous copy in Spanish: *"El dino se comió esta página"* + subtitle
- **Button:** Existing portfolio button style — `bg-accent text-black font-bold rounded-2xl px-6 py-3` — text: *"Volver al inicio"*
- **No external images, no JS runtime**
